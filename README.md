## How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/cp-solutions.git
   cd cp-solutions
   ```

2. **Compile and run C++ solutions:**
   ```bash
   g++ -o solution filename.cpp
   ./solution < input.txt
   ```

3. **Run Python solutions:**
   ```bash
   python3 filename.py < input.txt
   ```
